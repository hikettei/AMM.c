# AMM.cpp 

> AMM = Approximated Matrix Multiplication

### Motivation

- Dependency-Free
- Tiny
- Multi-Platform
- Simple API (callable via FFI)
- Fast
- The best platform for experiments
- Supporting Lowert Bits (2 ~ 8bits integers)

### Supported Architectures

- [ ] arm64
- [ ] x64 (AVX2, AVX512)
- [ ] CUDA
- [ ] Metal
